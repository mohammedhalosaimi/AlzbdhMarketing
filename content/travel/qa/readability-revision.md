# Readability Revision QA

Updated: 2026-06-13 Asia/Riyadh

Feedback applied:
- Preserve the real hero photo as the main background so each post stays visually distinct and catchy.
- Use lighter global overlays only for brand mood; do not flatten the full image into a solid dark backdrop.
- Put contrast directly behind text blocks, stats, and recommendation lines so Arabic stays readable on busy photos.
- Highlight the destination/place name with the secondary gold brand accent.
- Removed variable/low-value stat fields from the design: travel no longer shows duration or type; place cards no longer show type in the stat bar.
- Travel stat bars now focus on stable useful facts: budget and visa.
- Place stat bars now focus on useful discovery facts: Zbdh score, city, and price.

Live posts already published are not changed retroactively; this updates the local content pack for future posting.
